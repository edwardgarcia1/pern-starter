# PERN starter (Postgres, Express, React, Node + Typescript, Vite, shadcn/ui)

## /api

This is a template for a new Express project with TypeScript. Authentication with JWT and bcrypt. Authorization with CASL.

## /web

This is a template for a new Vite project with React, TypeScript, and shadcn/ui.

### Changing themes

Change src/index.css manually. Look for the following entries:

- primary (Theme color)
- secondary (Secondary color)
- background (Background color)
- foreground (Foreground color)
- muted (Muted color)
- muted-foreground (Muted foreground color)
- accent (Accent color)
- accent-foreground (Accent foreground color)
- popover (Popover color)
- popover-foreground (Popover foreground color)
- border (Border color)
- input (Input color)
- ring (Ring color)

## Adding components

Run the following command to add a component:

```bash
npx shadcn-ui@latest add [component-name]
```

## Adding icons

Run the following command to add an icon:

```bash
npx shadcn-ui@latest add [icon-name]
```
